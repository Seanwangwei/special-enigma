Safety
Draft-by-default prevents accidental mass sends. "Send Immediately" requires explicit selection each time.
Clarity
Step indicators show progress. Every action has a description. Color-coded status throughout.
Recovery
Per-student failure isolation. Errors tab in results. Full audit log written to .xlsx.
